Site

Snakebite is a browser game platform. It features the popular Snake game inspired in the old Nokia 3310 ranging only in HTML.

With recent technologies such as WebGL and WebAssembly, the browser has become an extremely powerful gaming platform. Amazing 3D graphics and near-native level performance are becoming possible without the need for downloads, apps, or platform-specific development. We are an innovative company at the forefront of this market, and are developing tools to help game developers to fully utilize it.
